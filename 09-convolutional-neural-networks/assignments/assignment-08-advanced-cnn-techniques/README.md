# Assignment 08 — Advanced CNN Techniques

This folder belongs to the eighth coding assignment from the course.

## Assignment focus
This assignment is focused on building and understanding a Convolutional Neural Network for image classification using the CIFAR-10 dataset.

## Main tasks included
- design and implement a CNN architecture from scratch
- use the CIFAR-10 dataset for multi-class image classification
- include at least two convolutional layers with ReLU activation
- include at least two max-pooling layers for spatial downsampling
- flatten the learned feature maps before classification
- add fully connected layers for final prediction
- use a 10-class softmax output layer
- compile, train, and evaluate the CNN using a complete deep learning workflow
