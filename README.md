# 🧩 Assignment Strategy

Coding assignments and research assignments are **not stored in one separate global folder**.

Instead, they are placed **inside the topic they belong to**, so the repository shows a natural progression:

> studied the concept → practiced it hands-on → completed the related assignment → documented the outcome

# 🧪 Practice Notebook Strategy

My practice notebooks are being treated as **portfolio evidence**, not as files to upload blindly.

Before they are placed into this repository, they should be:

- reviewed for secrets and credentials
- cleaned where necessary
- renamed consistently
- mapped to the correct topic folder
- supported with README context so each notebook has meaning

The goal is to keep the repository:

- safe
- clean
- easy to understand
- strong as a public portfolio
