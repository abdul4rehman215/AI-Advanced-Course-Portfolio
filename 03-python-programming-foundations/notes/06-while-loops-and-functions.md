# While Loops and Functions

This note covers two important ideas: repetition based on conditions and reusable blocks of logic.

## While Loops
A `while` loop runs as long as its condition remains true. Unlike a `for` loop, which is often used when iterating through a known sequence, a `while` loop is useful when the stopping point depends on a changing condition.

This topic helped me understand:
- how condition-driven loops work
- why counters need updating
- how infinite loops can happen if the condition never changes

That last point is important because a badly designed loop can consume resources and break a program.

## Functions
Functions allow logic to be grouped into reusable units. Instead of rewriting the same code multiple times, a function can be defined once and called whenever needed.

I practiced the key concepts:
- defining a function with `def`
- calling the function
- passing parameters
- returning results
- understanding local and global scope

This was one of the most useful Python topics because functions make code cleaner, easier to maintain, and easier to scale.

The biggest takeaway from this note is that functions move programming from simple scripting toward reusable design. They are one of the most important tools for writing cleaner code, and they become essential in later AI and ML notebook work.
