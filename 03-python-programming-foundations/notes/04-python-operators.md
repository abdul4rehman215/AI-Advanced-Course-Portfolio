# Python Operators

This note covers the operator-focused part of the course. Operators are important because they are used everywhere in logic building, calculations, comparisons, and control flow.

## Arithmetic Operators
These are used for basic calculations:
- addition
- subtraction
- multiplication
- division
- modulus
- exponentiation

The modulus operator was especially useful because it helps with patterns such as checking even and odd numbers.

## Comparison Operators
These compare values and return `True` or `False`. They are the base of conditional logic.

Examples include:
- equal to
- not equal to
- greater than
- less than
- greater than or equal to
- less than or equal to

A simple but important distinction here is the difference between assignment (`=`) and equality comparison (`==`).

## Logical Operators
Logical operators allow multiple conditions to be combined.

- `and` requires both conditions to be true
- `or` requires at least one condition to be true
- `not` reverses the condition

These are very useful in real decision-making logic.

## Membership Operators
Membership operators check whether something exists inside a collection.

- `in`
- `not in`

This becomes useful when checking input, lists, or dictionary keys.

The main lesson from this topic is that operators are not isolated syntax features. They are the tools used to build real program logic. Once I understood operators better, later topics like if statements and loops started making much more sense.
