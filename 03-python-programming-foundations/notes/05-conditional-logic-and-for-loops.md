# Conditional Logic and For Loops

This note covers the part of the course where Python starts becoming dynamic and decision-based.

## Conditional Statements
Conditional statements allow the program to choose a path based on a condition.

The course covered:
- `if`
- `if-else`
- `if-elif-else`

These structures are essential because they allow a program to respond differently depending on input or state.

An important detail here is indentation. In Python, indentation is part of the syntax. It is not just for visual style. If indentation is incorrect, the program logic breaks.

Another useful point was type awareness when taking input. Input often arrives as text, so sometimes it needs conversion before comparison or calculation.

## For Loops
For loops are used to repeat actions across a sequence. This removes the need to repeat code manually.

I practiced using loops with:
- lists
- strings
- ranges

The idea of iteration became clearer here. A loop goes item by item through a sequence and performs a repeated action.

The course also used practical examples like finding averages and checking even or odd values. These examples made it easier to understand how loops and conditionals work together.

This note was important because it showed how Python moves from static statements into controlled program flow. Once conditions and loops are understood, more meaningful problem solving becomes possible.
