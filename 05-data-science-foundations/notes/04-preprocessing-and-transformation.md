# Preprocessing and Transformation

This note covers the part of the course where raw data is turned into analysis-ready or model-ready data.

The course introduced several important preprocessing ideas:

## Data Cleaning
Cleaning involves handling issues such as:
- missing values
- inconsistent records
- noisy entries
- invalid values

## Data Integration
Sometimes data comes from multiple sources. Integration is the process of combining that information into one usable dataset.

## Data Reduction
Not every feature or record needs to stay in the final dataset. Reduction helps remove unnecessary complexity and can improve efficiency.

## Feature Engineering
Feature engineering creates or reshapes features so that the data becomes more useful for analysis or modeling.

## Discretization
This means converting continuous values into categories or bins when that makes interpretation or modeling easier.

## Normalization / Scaling
Scaling helps bring numeric values into a more comparable range so that large-value features do not dominate the model unfairly.

The course made preprocessing feel very important because poor preprocessing can damage model performance even when the model itself is strong.

My biggest takeaway is that preprocessing is where analytical judgment matters a lot. It is not only about applying techniques. It is about deciding what the data needs before modeling.
