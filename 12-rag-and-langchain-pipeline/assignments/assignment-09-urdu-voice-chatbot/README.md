# Assignment 09 — Urdu Voice Chatbot

This folder belongs to the ninth coding assignment from the course.

## Assignment focus
This assignment is focused on building an intelligent Urdu voice chatbot using Streamlit, LangChain, an LLM API, retrieval-augmented generation, and audio input-output workflows.

## Main tasks included
- build a Streamlit chatbot interface for real-time Urdu interaction
- accept Urdu voice input from the user
- convert speech to text for model processing
- integrate an LLM API through LangChain
- use retrieval-augmented generation with PDF-based knowledge sources
- generate context-aware chatbot responses instead of plain generic answers
- return both text output and audio output in Urdu
- manage project dependencies and application structure properly
