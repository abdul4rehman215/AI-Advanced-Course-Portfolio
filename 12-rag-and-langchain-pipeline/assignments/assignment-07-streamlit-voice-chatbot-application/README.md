# Assignment 07 — Streamlit Voice Chatbot Application

This folder belongs to the seventh coding assignment from the course.

## Assignment focus
This assignment is focused on building a Streamlit-based voice chatbot application that can receive Urdu voice input, process it, and return speech output in Urdu.

## Main tasks included
- build a Streamlit interface for Urdu voice interaction
- record or receive voice input in Urdu
- convert Urdu speech into text using speech-to-text processing
- generate a response based on the recognized input
- convert the response text back into Urdu speech
- display or play the generated speech output through the application
- package the project as a Python application with dependency tracking
