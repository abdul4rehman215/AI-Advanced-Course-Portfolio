# Research Assignment — Study HTML DOM Structure

This folder contains the HTML DOM structure study assignment completed for the course.

## Research focus
This assignment is centered on understanding the Document Object Model (DOM), how browsers represent HTML as a tree structure, and why DOM knowledge matters for modern web and AI-related application workflows.

## Main areas covered
- what the DOM is
- how HTML becomes a DOM tree inside the browser
- element nodes, text nodes, and attributes
- parent, child, and sibling relationships
- DOM traversal and event handling
- the relationship between HTML source code and the live DOM
- JavaScript interaction with the DOM
- practical examples and real-world importance of DOM knowledge
