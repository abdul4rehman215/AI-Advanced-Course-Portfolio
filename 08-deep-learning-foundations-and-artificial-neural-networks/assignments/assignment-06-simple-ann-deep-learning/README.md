# Assignment 06 — Simple ANN (Deep Learning)

This folder belongs to the sixth coding assignment from the course.

## Assignment focus
This assignment is centered on core artificial neural network concepts and a first practical dense-network implementation using the MNIST handwritten digits dataset.

## Main tasks included
- explain the basic structure and working of a simple Artificial Neural Network (ANN)
- describe the role of neurons, layers, weights, and biases
- explain how information flows through an ANN
- explain activation functions and mention commonly used examples
- build a fully connected neural network for handwritten digit recognition using MNIST
- use an input layer of size 784 and hidden layers with ReLU activation
- train and evaluate the model using a proper deep learning workflow
