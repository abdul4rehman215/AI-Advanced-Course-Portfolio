# Why Deep Learning Over Traditional Machine Learning

This note covers the lecture where the course explained why deep learning became so important in modern AI.

A major point was that conventional machine learning has limits. Classical models can perform well on structured tabular data, but they struggle when the input becomes unstructured, high-dimensional, or too complex for hand-crafted features. Images, speech, and large-scale text are good examples of that difficulty.

The lecture emphasized that classical ML often depends heavily on feature engineering. In those workflows, a human expert has to decide which features should be extracted and how they should be represented. That is time-consuming and also restrictive because the model can only learn from what the human decided to expose.

Deep learning changes that picture. Neural networks are able to learn useful representations directly from raw or minimally processed data. That means they can automatically extract patterns from images, audio, and text without needing the same level of manual engineering.

Another strong point from the lecture was scalability. Classical models often plateau. Deep learning systems, by contrast, can improve as more data, more compute, and better architectures become available. That explains why deep learning expanded so strongly in the modern AI era.

My biggest takeaway is that deep learning is not just “a more advanced algorithm.” It is a different learning strategy for dealing with complexity, scale, and unstructured information.
